Novità nella v. 6.5 di APDFC:

- Aggiunte notifiche lato desktop all'aggiunta delle pagine e alla creazione del file PDF
- Aggiunta la possibilità di eliminare le singole immagini dal PDF prima della sua creazione con il tasto destro
- Aggiunta la possibilità resettare la creazione del file nel menu superiore
- Aggiunta la compressioni delle singole immagini e la possibilità di modificarne il fattore tramite slider
- Le immagini compresse vengono gestite nella cartella TMP del sistema
- Alcuni bug minori sono stati risolti
- Alcuni problemi grafici sono stati risolti
